# AI-chess-experiment
An experiment in chess AI using Monte-Carlo simulation method.<br/>
Required Python 3.3+<br/>
How to run:
```
  cd chess
  python parallel-runner.py
```
At the end of the simulation, the result will be printed out on the console:<br/>
112411131|322<br/>
112111314|210<br/>
114111132|310<br/>
412111311|221<br/>
114111312|300<br/>
...

First value is the board state, and second value is the action that provides best outcome.
