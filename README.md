# AI-chess-experiment
An experiment in chess AI using Monte-Carlo simulation method.<br/>
Required Python 3.3<br/>
How to run:
```
  cd chess
  python parallel-runner.py
```
After the simulation, output files will be generated with the following name:<br/>
chess-parallel-0<br/>
chess-parallel-1<br/>
...<br/>
chess-parallel-9<br/>
